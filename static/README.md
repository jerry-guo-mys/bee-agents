# Web UI 增强功能

Bee 的 Web UI 是一个现代化、功能丰富的单页应用，基于原生 JavaScript 和 CSS 构建。

## 已实现功能

### 核心功能
- 流式响应显示
- 会话管理（创建、切换、删除、搜索）
- 深色/浅色主题切换（支持系统偏好）
- 文件拖拽上传
- 图片粘贴支持
- 代码高亮与行号
- 消息复制功能

### 键盘快捷键
- `Enter` - 发送消息
- `Shift+Enter` - 换行
- `/` - 聚焦输入框
- `Ctrl+L` - 新对话
- `Ctrl+K` - 搜索对话
- `Ctrl+F` - 当前对话内搜索
- `Esc` - 停止生成 / 关闭搜索

### UI 增强
- 骨架屏加载动画
- Toast 通知系统
- 连接状态指示器
- 字体大小调节
- 滚动到底部按钮
- 消息时间戳
- 响应式移动端适配

### 对话管理
- 会话搜索
- 会话重命名（双击标题）
- 会话导出/导入
- Token 使用统计

## 技术特性

- 无需构建步骤，纯原生实现
- CSS 变量主题系统
- Intersection Observer 性能优化
- 防抖滚动处理
- 无障碍支持（ARIA 标签、焦点管理）
- 减少动画偏好支持
