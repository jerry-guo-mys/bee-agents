# 英语写作模板

## 学术论文结构

### Introduction
- Background
- Research gap
- Thesis statement

### Body
- Topic sentence
- Supporting evidence
- Analysis
- Transition

### Conclusion
- Restate thesis
- Summary
- Implications

## 商务邮件模板

```
Subject: [Clear and Specific]

Dear [Name],

[Opening - Purpose of email]

[Body - Key points, numbered if multiple]

[Closing - Action items / Next steps]

Best regards,
[Your name]
```

## 常用句型

- **引入观点**: It is widely acknowledged that...
- **对比**: In contrast, ... However, ...
- **举例**: For instance, ... A case in point is...
- **总结**: In conclusion, ... To sum up, ...
