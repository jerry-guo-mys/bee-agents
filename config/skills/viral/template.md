# 爆款文章模板

## 标题模板

1. **数字型**：X个方法让你...
2. **疑问型**：为什么...？
3. **对比型**：A vs B，到底哪个更...
4. **揭秘型**：99%的人不知道的...
5. **痛点型**：还在为...烦恼？

## 文章结构

### 开头（钩子）
- 痛点共鸣 / 热点切入 / 悬念引入

### 正文
- 分点阐述（3-5个要点）
- 每个要点配案例/数据
- 适当使用金句

### 结尾
- 总结升华
- 行动号召（CTA）

## 小红书适配

- 标题带 emoji
- 首图要吸睛
- 正文分段短小
- 结尾引导互动

## 公众号适配

- 开头 3 秒决定阅读
- 中间多用小标题
- 结尾引导转发/在看
